Visualization of Carbon Dioxide Emissions


Objective

This project aims to show Carbon dioxide (CO2) emissions from fossil fuels and industry since the early 1800 per year, worldwide, in the country, and by capita.


Design

The webpage follows an interactive slideshow, one of the narrative visualization structures from the CS416 Data Visualization class of UIUC. This allows users to navigate the pages and view changes in CO2 emissions by various factors.

Users can navigate the pages by clicking list items in the left sidebar. The main page is the Introduction, and the sidebar allows users to navigate through all the pages in any order and quickly return to the first page.

The webpage is designed to best fit the desktop. When the window size of the browser changes, users can use the refresh button to redraw graphs on each page in order to fit them into a new window size.


Data

The CO2 dataset includes annual carbon dioxide emissions for each country in the world, including emissions per capita and fuel types such as oil, coal, natural gas, and etc. The original data can be downloaded from the Global Carbon Project.
